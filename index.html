📢 EarthquakeTsunamiYomiageとは｜地震・津波速報を読み上げる防災アプリ

地震情報・津波情報・緊急地震速報を自動で読み上げる無料ソフトウェアです。

🔧 主な機能と使い方｜直感的なUIと高精度の読み上げ

画面 説明
トップ画面
受信した地震・津波情報の一覧が表示されます。ここに表示された文章が読み上げられ、新しい情報が来ると自動的にスクロール更新されます。
音声設定画面
音声ごとの読み上げの有無を切り替えることができます。ここに記載されているファイル名と対応した情報は、READMEまたは当サイト下部「音声ファイルの名前対応表」からでも確認可能です。
棒読みちゃん連携設定画面
「棒読みちゃん」との接続設定を行う画面です。使用するポート番号の変更や、HTTPサーバー機能の有効／無効を切り替えることができます。なお、棒読みちゃん側の設定変更後は再起動が必要です。
読み上げ設定画面
各情報ごとに受信時に、その情報を読み上げるか読み上げないかの設定ができます。この設定はv2.1.0以降に追加されています。是非最新版をご利用ください！

⚙️ 設定について

棒読みちゃん連携設定

  1. 棒読みちゃんの「アプリケーション連携」→「HTTP連携」→「ポート番号」を変更
  2. 「ローカルHTTPサーバー機能を使う」が False になっていたら True に変更
  3. 設定後は 棒読みちゃんを再起動 してください

🧾 config.jsonについて

config.json を編集することで以下の設定が可能です。"" で囲まれた文字列のみを変更してください。

  • 音声ファイルのフォルダパス
  • 各種音声ファイル名
  • 津波メッセージの文言
  • 地震情報のラベル名（例：遠地地震情報）
  • WebSocketの受信先URL（P2P/Wolfx）

📂 インストール方法

  1. 上記のダウンロードリンクから ZIP ファイルを取得
  2. ZIP を解凍し、フォルダ内の EarthquakeTsunamiYomiage.exe を実行
  3. 初回起動時にセキュリティ警告が出た場合は「詳細情報」→「実行」を選択
