EarthquakeTsunamiYomiage

地震情報・津波情報・緊急地震速報を自動で読み上げるソフトウェアです。

🔧 主な機能

画面 説明
緊急地震速報・津波情報などを一括表示。
音声ごとのON/OFF、棒読みちゃんの声・音量・速度・音程の変更が可能。
棒読みちゃんとの連携設定画面。ポート番号の変更が可能。

⚙️ 設定について

棒読みちゃん連携設定

  1. 棒読みちゃんの「アプリケーション連携」→「HTTP連携」→「ポート番号」を変更
  2. 「ローカルHTTPサーバー機能を使う」が False になっていたら True に変更
  3. 設定後は 棒読みちゃんを再起動 してください

🧾 config.jsonについて

config.json を編集することで以下の設定が可能です。"" で囲まれた文字列のみを変更してください。

  • 音声ファイルのフォルダパス
  • 各種音声ファイル名
  • 津波メッセージの文言
  • 地震情報のラベル名（例：遠地地震情報）
  • WebSocketの受信先URL（P2P/Wolfx）

📢 配信での利用について

  • クレジット表記は 不要 です。ご自由にご利用ください。

🔗 データ取得元

  • P2P地震情報 API v2
  • Wolfx API

📂 インストール方法

  1. 上記のダウンロードリンクから ZIP ファイルを取得
  2. ZIP を解凍し、フォルダ内の EarthquakeTsunamiYomiage.exe を実行
  3. 初回起動時にセキュリティ警告が出た場合は「詳細情報」→「実行」を選択
